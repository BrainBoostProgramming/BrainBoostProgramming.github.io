First let's take a look at for loops. They are pretty nice. Here's a picture that I took from the internet:

After the keyword for there are three things going down.

First: int i = 0; is saying that within the scope of this loop there shall exist an integer variable i and it will start at 0.

Second: i < 10; is the condition for the loop to continue under. Once i is greater than or equal to 10, the loop will end.

Third: i++ is increasing i by one each time the loop completes.

Some examples:

//print out numbers from 0 to 1000
for (int i = 0; i < 1001; i++) {
    System.out.println(i);
}

//print out numbers from 50 to 25
for (int i = 50; i > 24; i--) {
    System.out.println(i);
}

//print out 0-100 in increments of 5
for (int i = 0; i < 101; i += 5) {
    System.out.println(i);
}

//print out each character of a string on a new line
String hello = "Hello World";
for (int i = 0; i < hello.length(); i++) {
    System.out.println(hello[i]);
}

There's five questions and they all involve the java.util.Random class. Here's a program that outputs 100 random numbers between 1 and 25:

import java.util.Scanner;
import java.util.Random;

class GenerateNumbers {
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        Random generaterandom = new Random();

        int num;

        for (int i = 0; i < 100; i++) {
            num = generaterandom.nextInt(25) + 1;
            System.out.println(i + " " + num);
        }
    }
}
