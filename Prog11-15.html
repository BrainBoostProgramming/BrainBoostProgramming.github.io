Switch statements do pretty much the same thing as an if/else if/else block but are easier to make sense of and use in some cases. Take this example from the Java tutorial on switch statements:

public class SwitchDemo {
    public static void main(String[] args) {

        int month = 8;
        String monthString;
        switch (month) {
            case 1:  monthString = "January";
                     break;
            case 2:  monthString = "February";
                     break;
            case 3:  monthString = "March";
                     break;
            case 4:  monthString = "April";
                     break;
            case 5:  monthString = "May";
                     break;
            case 6:  monthString = "June";
                     break;
            case 7:  monthString = "July";
                     break;
            case 8:  monthString = "August";
                     break;
            case 9:  monthString = "September";
                     break;
            case 10: monthString = "October";
                     break;
            case 11: monthString = "November";
                     break;
            case 12: monthString = "December";
                     break;
            default: monthString = "Invalid month";
                     break;
        }
        System.out.println(monthString);
    }
}

This could also be achieved like so:

int month = 8;
if (month == 1) {
    System.out.println("January");
} else if (month == 2) {
    System.out.println("February");
}
// and so on

One thing to note is that without the break; after each case in the switch, the code will keep running through the cases.

    The program should run in a loop and continue until a value of 0 is entered for the days overdue.

    Here is an example program that you can look at for clues:

    import java.util.Scanner;

class Switcher {
    public static void main(String[] args) {

        String color;
        char choice;
        String message = "Color is";

        Scanner scan = new Scanner(System.in);

        System.out.print("Enter a color letter: ");
        color = scan.nextLine();

        color = color.trim(); //remove any leading or trailing whitespace
        color = color.toUpperCase(); //convert the whole string to uppercase
        choice = color.charAt(0); //get the first of the string

        switch (choice) {
            case 'R':
                message = message + " red";
                break;
            case 'O':
                message = message + " orange";
                break;
            case 'Y':
                message = message + " yellow";
                break;
            case 'G':
                message = message + " green";
                break;
            case 'B':
                message = message + " blue";
                break;
            case 'V':
                message = message + " violet";
                break;
            default:
                message = message + " unknown";
        }

        System.out.println(message + " " + choice);
    }
}
