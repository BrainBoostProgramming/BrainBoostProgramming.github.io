This section is supposed to be about Java Applets. However, Java Applets were deprecated ages ago in favor of Flash which in turn has been deprecated in favor of JS/HTML5. So we are going to do that. JS is JavaScript and HTML is Hyper Text Markup Language. These languages, along with CSS (Cascading Style Sheet), make up the front end of the internet.

Despite the name, JavaScript and Java are entirely different. We won't get too into JavaScript, but here are some important differences:

Creating variables:

//In java, you must declare the type of variable.
int x = 5;

//In JS, you don't. This is both a good and bad thing.
//The keyword 'let' is how you tell JS to create a new variable.
let x = 5;

Creating functions:

//in Java there's a bunch of stuff going on when creating a function.
public static void main(String[] args) {
    //do something
}

//simpler in js
function Main(args) {
    //do something
}

Note that in JS you don't usually need a main function or class.
