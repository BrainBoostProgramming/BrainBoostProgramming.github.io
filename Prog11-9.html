Assignment 9

From now on you should be adding comments to your code. For all the programs you work on, include comments that explain what each section is doing.

Do:

  • Add comments above each section to briefly describe what each section is doing.
  • Be descriptive.
  • Use good variable names that describe what each variable is for.

Do not:

  • Add comments to each line. This is not necessary and looks messy.
  • Write a book. Keep the length of your comments to the minimum required to explain a section.

    Here's the (buggy) solution to this question if you get stuck. Make sure you have it working before handing in:

    import java.util.Scanner ;

public class Adder {

    public static void main (String args[]) {

        Scanner scan = new Scanner(System.in);

        int n, tot, input,c;

        tot = 0;

        System.out.println("How many integers will be added?");

        n = scan.nextInt();

        c = 0;//you are going to count so start at zero

        //keep going while your counter is less than the number of times you want to print
        while (c < n) {

            System.out.println ( "Enter an integer:" );

            input = scan.nextInt ();

            tot = tot + input;

            c++;//this means add 1 to c you could do c=c+1; instead
        }

        System.out.println ( "The sum is " + tot + ".");
    }
}
