Programming 11

Assignment 1

  1. Download and install the latest JDK (currently version 9).

  2. Download and install BlueJ.

  3. Clone the course skeleton to your computer.

  4. Cool, now open BlueJ!

  5. In BlueJ click Project, then Open Project...

  6. Select the 'Assignment1' folder as the project to open.

  7. Double click the class 'HelloWorldApp' to open it in the BlueJ editor.

  8. Copy and paste following code into the editor.

    /**
* The HelloWorldApp class implements an application that simply
* displays "Hello World!" to the standard output.
*/
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

  9. Click on 'Compile' in the editor window. This will compile the human-readable source code into byte code that can be executed by the Java Virtual Machine (more on what that is later).

    You should hopefully see this message on the bottom left of the window.

  10. Back in the main BlueJ window, right click the "HelloWorldApp" rectangle and click 'void main(String[] args)'.

  11. Hit 'OK' in the window that pops up.

  12. Wow! It did the thing.
