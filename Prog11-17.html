Programming 11

Assignment 17

Design for a User Assignment

A friend (Charlie) has asked you to create a system to store information about a basketball card collection. Complete the questions below to show how you would help. You will need to discuss how you would use the tools learned in this course to get full credit for this assignment (you cannot tell your friend to use another database type program). Note: you DO NOT need to create the program for this assignment, just discuss it.

  1. Write a problem statement for this scenario. What is a problem statement? Do a web search or try this link.
  2. You will need more information to create this system. List 2 questions you would ask Charlie.
  3. Create a simple flow chart to show how your program might work. Have a look at this link for more information about flow charts.
  4. Now that you have an idea how your program might run, think of two more questions you would ask Charlie to help you get a database that is closer to what Charlie would want.
