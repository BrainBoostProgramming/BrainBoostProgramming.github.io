Boom Chapter 12. Whole buncha stuff in there, don't worry if it doesn't make sense at first. Pretty much stuff is input into the computer and we can use Java to figure out what that stuff is (spoiler: it's mostly keyboard strokes) and do cool stuff with it.

Just to start let's get this program running with BlueJ:

import java.util.Scanner;
class Echo {
    public static void main (String[] args) {
        String inData;
        Scanner scan = new Scanner(System.in);
        System.out.println("Enter the data:");
        inData = scan.nextLine();
        System.out.println("You entered:" + inData);
    }
}

All this program does is prompts the user for some input, then outputs it back to them. Let's go over it line by line.

import java.util.Scanner;

This line tells the compiler that our program is going to include the Scanner package. The Scanner class is what will allow us to read characters entered into the console.

class Echo {
    public static void main (String[] args) {

We declare the class Echo and give it a main method.

String inData;

We declare a variable of type String named inData. We don't give it an initial value.

Scanner scan = new Scanner(System.in);

This line creates a new instance of the Scanner class called scan. This particular Scanner instance is going to be used to read the system input, so we pass in System.in.

System.out.println("Enter the data:");

Prompts the user to enter some data.

inData = scan.nextLine();

When the computer hits this line, execution is paused until the user hits the enter key. Once the enter key is pressed, the scan object scans whatever was input up to the point the enter key was pressed (which would start the next line). That scanned text is then stored in the inData variable we declared at the start of the program.

System.out.println("You entered:" + inData);

Lastly, we print out the value of the inData variable.
