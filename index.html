Intro and Github

Sign up for an account on Github and download/install the Github client. Log in to the client once it's installed.

Go to this repository and click the 'fork' button.

You'll be prompted for where to fork it. Choose your Github account. You should then see a page that looks like this:

Open up the Github desktop application and hit Ctrl+Shift+O. This will open the menu for cloning a repository. Choose your Java11Skeleton fork and the folder you want to put it in (I recommend having a 'repos' folder in your user folder).

The skeleton is now on your computer and you are ready to start working!
